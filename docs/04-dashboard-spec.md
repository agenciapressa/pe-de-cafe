# Dashboard Specification — Um Pé de Café

## Objetivo
Definir a estrutura conceitual do dashboard da futura plataforma.

## Módulos principais

### 1. Visão geral da propriedade
- nome da fazenda
- localização
- área total
- área de café
- status do onboarding

### 2. Indicadores climáticos
- estimativa de carbono
- baseline
- status MRV
- evolução por período

### 3. Governança e conformidade
- documentação enviada
- auditorias
- pendências
- elegibilidade

### 4. Pool regional
- propriedades participantes
- hectares totais
- status agregado
- diluição de custos

### 5. Relatórios
- exportação PDF
- visão executiva
- visão técnica
- trilha de auditoria

## Perfil de usuários
- produtor
- gestão interna
- auditor / parceiro técnico
- investidor / parceiro institucional

## Status
Documento inicial conceitual, sujeito a expansão para wireframe e especificação funcional futura.
